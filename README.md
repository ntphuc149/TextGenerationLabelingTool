Theme customization:
- Create a folder name: .streamlit
- Create file name 'config.toml' inside folder '.streamlit'
- Paste this content into 'config.toml':

[theme]\n
primaryColor="#8e2929"\n
backgroundColor="#1f1f1f"\n
secondaryBackgroundColor="#464646"\n
textColor="#ce9178"\n
font="serif"
